Leadership and Philosophy:

My Leadership Approach
The administration needs to encourage all workers to work in teams in order to achieve a set goal. Based on the learning organization idea proposed by Peter Senge, I build environment in which each-member-valued approach is fostered.
Example in Action When there was a global IT upgrade, I chaired several workshops so that all stake holders would appreciate what the actual project was all about. With creative communication and cooperative approach to work we handled conflicts and we were able to finish the upgrade two weeks earlier. Core Values
• Inclusivity: Seek for all possible angles because the best solution is one from all sides.
• Transparency: Thus, to strengthen the relationship between the organization’s two key players, it important to employ transparent communication techniques.
• Growth-Oriented Leadership: Teach the members of the mentor team how exactly to realize their personal goals and improve the performance.
